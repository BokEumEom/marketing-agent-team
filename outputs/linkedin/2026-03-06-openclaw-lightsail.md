# AWS OpenClaw on Lightsail — 내 서버에서 돌아가는 AI 에이전트

**채널:** LinkedIn
**날짜:** 2026-03-06
**캠페인:** 2026-03-launch
**상태:** draft

---

AI를 쓰고 싶은데 데이터 보안이 걱정됐다면, AWS가 해답을 내놨다.

Amazon Lightsail에 OpenClaw가 추가됐다. 오픈소스 AI 에이전트인데, 특이한 점이 하나 있다. 내 서버에서만 돌아간다. 외부로 데이터가 나가지 않는다.

기능은 단순하지 않다. WhatsApp, Discord, Telegram 같은 메신저에 연결해서 이메일 정리, 웹 검색, 파일 관리를 자율적으로 처리한다. 질문에 답하는 챗봇이 아니라 실제로 일을 대신 처리하는 에이전트다.

설정도 복잡하지 않다. Lightsail 콘솔에서 블루프린트를 선택하면 AI 모델(기본값: Amazon Bedrock + Claude Sonnet 4.6)까지 자동으로 연결된다. 서버 세팅이나 모델 API 연동을 직접 할 필요가 없다.

기업 입장에서 보면 의미가 다르다. 내부 문서, 고객 데이터를 외부 AI 서비스에 올리는 것에 대한 우려가 많았다. OpenClaw는 그 문제를 구조적으로 해결한다. 데이터가 내 인프라 안에서만 처리된다.

AI 도입의 가장 큰 장벽 중 하나가 데이터 보안이었다. 그 장벽이 낮아지고 있다.

# Go-to-Market 전략

## 목표

**90일 안에** Claude Code + AI 에이전트 관심 커뮤니티에서 인지도 구축.

## 핵심 메시지

"개발자가 마크다운 파일만으로 AI 마케팅팀을 만들 수 있다"

## 채널 전략

### X/Twitter (메인 채널)
- **목표**: 팔로워 +500, 주요 포스트 노출 10만+
- **빈도**: 주 3-4회
- **콘텐츠 유형**: 빌드 로그 스레드, 숫자 중심 인사이트, 질문/토론
- **전술**: Claude Code 관련 계정, AI 자동화 계정과 교류

### LinkedIn (전문가 채널)
- **목표**: 아티클 조회수 1만+
- **빈도**: 주 1-2회
- **콘텐츠 유형**: 심층 아티클, 빌드 과정 공유
- **전술**: 개발자/마케터 그룹에 공유

### Threads (커뮤니티 채널)
- **목표**: 팔로워 +300
- **빈도**: 주 2-3회
- **콘텐츠 유형**: 짧은 인사이트, X 포스트 재활용
- **전술**: Instagram 연계 노출

### Maily (뉴스레터 — 깊이 채널)
- **목표**: 구독자 500명
- **빈도**: 격주 (2주에 1회)
- **콘텐츠 유형**: AI 마케팅 자동화 깊이 있는 내용
- **전술**: X/LinkedIn에서 구독 유도

## 콘텐츠 필러 (4가지 주제)

1. **빌드 과정** — 어떻게 만들었는지 (에이전트 설계, 파일 구조, 실패 사례)
2. **실제 결과** — 수치로 보는 효과 (비용, 노출, 참여율)
3. **기술 인사이트** — Claude Code, MCP, 에이전트 패턴
4. **아이디어 공유** — AI 마케팅의 가능성과 한계

## 90일 로드맵

### 1단계 (1-30일): 기반 구축
- 채널 설정 및 프로필 최적화
- 런칭 시리즈: "AI 마케팅팀을 만든 방법" 3부작
- 뉴스레터 첫 호 발행

### 2단계 (31-60일): 성장
- 주 3회 이상 X 포스팅 유지
- LinkedIn 아티클 2편 발행
- 성과 데이터 공개 포스트

### 3단계 (61-90일): 최적화
- 성과 분석 기반으로 채널/형식 집중
- 구독자 확대 캠페인
- 커뮤니티 교류 확대

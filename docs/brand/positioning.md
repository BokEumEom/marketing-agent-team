# 제품 포지셔닝

## 제품명

AI 마케팅팀 에이전트 (claude-marketing-team-agent)

## 한 줄 설명

Claude Code로 4명의 AI 마케팅 팀원을 만들어 마케팅을 자동화하는 오픈소스 시스템.

## 포지셔닝 스테이트먼트

> 마케팅 예산이 없거나 팀이 없는 개인 개발자/스타트업을 위한,
> Claude Code 에이전트 팀으로 X, LinkedIn, Threads, 뉴스레터를 자동 운영하는 시스템.
> 별도 플랫폼이나 코드 없이 마크다운 파일 15개로 완성된다.

## 타겟 고객

### 주요 타겟

- **1인 개발자/소규모 팀**: 사이드 프로젝트나 SaaS를 운영하지만 마케팅할 시간이 없는 개발자
- **Claude Code 사용자**: 이미 Claude Code를 쓰고 있어 진입 장벽이 낮은 사람

### 부차 타겟

- **마케터**: AI 도구로 업무 자동화에 관심 있는 실무자
- **AI 에이전트 연구자**: 멀티에이전트 시스템 사례에 관심 있는 사람

## 핵심 가치 제안

1. **코드 없이 에이전트 팀 구성** — 마크다운 파일이 전부
2. **비용 효율** — 풀타임 마케터 대신 하루 $20-50 API 비용
3. **병렬 실행** — 여러 에이전트가 동시에 다른 채널 담당
4. **점진적 자율화** — 수동 → 반자율 → 완전 자율로 단계적 전환 가능

## 경쟁 포지셔닝

| 대안 | 차이점 |
|------|--------|
| 마케팅 SaaS (Buffer, Hootsuite) | 우리: 콘텐츠 생성까지 AI가 담당, 구독료 없음 |
| GPT-4 직접 사용 | 우리: 에이전트 팀 구조, 기억/학습 가능 |
| 마케팅 대행사 | 우리: 100배 저렴, 24시간 가동 |
| 직접 작성 | 우리: 시간 90% 절약 |

## 메시지 계층

- **Level 1 (헤드라인)**: Claude Code로 AI 마케팅팀을 만들었다
- **Level 2 (부제)**: 마크다운 15개 파일, 하루 $30, 4명의 AI 팀원
- **Level 3 (세부)**: CMO가 전략을 짜고, 콘텐츠 라이터가 글을 쓰고, 소셜 마케터가 배포하고, 뉴스레터 매니저가 발송한다
